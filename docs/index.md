# S.D.T. — Software Digital Thread

S.D.T. is a repo birth and repo baseline system for making serious repos easier for humans and AI to understand, continue, and hand off.

## What it does

- births new repos with a continuity floor
- baselines existing repos without pretending to invent missing product truth
- checks whether the SDT floor is present
- generates Markdown and HTML repo reports

## What is already proved

- the CLI installs and runs on the control host
- template validation exists and passes
- report-only baseline works
- repo report generation works
- a clean birth proof exists from template to first independent commit

## Start here

## What SDT is not

SDT is not the product itself.
It is the continuity, truth, handoff, and governance system around serious repos.
