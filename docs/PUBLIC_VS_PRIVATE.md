# Public vs protected

## What this repo is

This repo is the public limited subset of the broader SDT operating model.

## What is public here
- public bootstrap and baseline tooling
- public usage docs
- public example surfaces
- public report generation surfaces

## What stays protected elsewhere
- control-plane truth
- raw adoption audits
- raw operator playbooks
- repo registration truth
- protected product backplanes
- private runtime and proof materials

## Rule

Do not split one repo into public and private halves.
Use separate repos by role and sensitivity.
